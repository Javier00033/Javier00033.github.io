# Portafolio Web - Javier Gutiérrez Medina

## Descripción

Portafolio web profesional desarrollado para Javier Gutiérrez Medina, Ingeniero en Telecomunicaciones y Electrónica. El sitio está diseñado para ser extremadamente ligero, rápido y profesional.

## Características

- ✅ **Diseño Responsivo** - Optimizado para todos los dispositivos
- ✅ **Rendimiento Ultra-rápido** - Carga instantánea optimizada
- ✅ **SEO Optimizado** - Mejores prácticas de SEO implementadas
- ✅ **Accesibilidad** - Cumple con estándares de accesibilidad
- ✅ **Animaciones Suaves** - Interacciones fluidas y modernas
- ✅ **Lenguajes** - UI en inglés y español
- ✅ **Navegación Intuitiva** - Experiencia de usuario superior

## Tecnologías Utilizadas

- **HTML5** - Estructura semántica
- **Tailwind CSS** - Framework CSS utilitario
- **JavaScript Vanilla** - Sin dependencias pesadas
- **Google Fonts** - Tipografías optimizadas
- **Font Awesome** - Iconos vectoriales

## Estructura del Proyecto

```
/
├── index.html          # Página principal del portafolio
├── main.js            # JavaScript funcional y optimizaciones
├── README.md          # Este archivo
└── resources/         # Carpeta para recursos adicionales
    └── (archivos de imagen, documentos, etc.)
```

## Secciones del Portafolio

1. **Hero/Inicio** - Presentación principal con animaciones
2. **Experiencia** - Trayectoria profesional detallada

4. **Habilidades** - Competencias técnicas y personales

## Optimizaciones de Rendimiento

- **CSS Minificado** - Tailwind CSS desde CDN optimizado
- **JavaScript Ligero** - Vanilla JS sin frameworks pesados
- **Imágenes Optimizadas** - Uso de iconos vectoriales
- **Lazy Loading** - Carga diferida de elementos
- **Intersection Observer** - Animaciones eficientes
- **Preload de Recursos** - Carga anticipada de recursos críticos

## Instalación y Uso

### Opción 1: Servidor Local

```bash
# Clonar o descargar los archivos
# Navegar al directorio del proyecto
cd portfolio-web

# Iniciar servidor local
python -m http.server 8000
# o
npx serve .

# Abrir en navegador
# http://localhost:8000
```

### Opción 2: Deployment Inmediato

El portafolio está listo para ser desplegado en cualquier servicio de hosting estático:

- **GitHub Pages** - Subir archivos a repositorio
- **Netlify** - Arrastrar y soltar los archivos
- **Vercel** - Importar desde GitHub
- **Firebase Hosting** - CLI de Firebase
- **AWS S3** - Hosting estático

## Personalización

### Colores
Los colores principales están definidos en las variables CSS en `index.html`:
```css
:root {
    --primary-color: #2563eb;
    --secondary-color: #1e40af;
    --accent-color: #3b82f6;
}
```

### Contenido
